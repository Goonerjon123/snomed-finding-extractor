# Regulatory Anchors

Checked planning anchors:

- MHRA Software and AI as a Medical Device guidance.
- MHRA guidance on Class I medical devices.
- NHS DCB0129 clinical risk management standard.
- NHS TRUD SNOMED CT UK Clinical Edition RF2 release channel.

The implementation is designed for a narrow clinician-confirmation workflow, but formal UK medical device classification must be confirmed before release. Intended-use wording, integration behaviour, and any move toward auto-coding may change the classification and evidence burden.
